# ACIS — Insurance Risk Analytics Week

## Overview
Project to analyze car insurance claims (Feb 2014 — Aug 2015) and identify low-risk segments.

## Structure
- `data/` — raw CSV files (gitignore large raw data if needed)
- `notebooks/` — EDA & analysis notebooks
- `src/` — reusable scripts and modules
- `.github/workflows/` — CI pipelines
- `README.md` — this file

## How to run
1. Create a virtualenv: `python -m venv .venv && source .venv/bin/activate`
2. Install: `pip install -r requirements.txt`
3. Run notebooks or scripts in `notebooks/`

## Deliverables
- EDA notebook (`notebooks/EDA_task1.ipynb`)
- 3 visualization PNGs / SVGs
- Interim report (PDF)
